📚 Book API

API em desenvolvimento para gerenciamento de livros, permitindo criar novos registros e consultar dados armazenados em um banco PostgreSQL.

O projeto está sendo atualizado continuamente desde o início, com melhorias, novas funcionalidades e refatorações.
